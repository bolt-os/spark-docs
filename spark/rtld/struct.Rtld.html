object: &'elf Elf<'elf>link_base: usizeimage_base: usize

Physical address at which the object’s image was loaded

image_size: usize

Total size of the image, in bytes

load_base: usize

Virtual address at which the object’s image was loaded

reloc_base: usize

Relocation slide applied to the object (general relocation/KASLR)

This value is the difference between the address at which the object was linked and the actual address at which is was loaded (load_base).
